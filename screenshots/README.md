# Скриншоты

Положите сюда PNG-скриншоты приложения с такими именами — они автоматически подхватятся в корневом README.md:

- `01_auth.png` — экран входа / регистрации
- `02_goals.png` — список эко-целей
- `03_stats.png` — вкладка «Статистика»
- `04_ai.png` — вкладка «AI-советник» с ответом Gemini
- `05_map.png` — карта эко-точек
- `06_profile.png` — профиль (XML + ComposeView)

Снять на эмуляторе: Android Studio → Running Devices → значок камеры. Или `adb exec-out screencap -p > screenshots/01_goals.png`.
